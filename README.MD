# BootStack - Linux Setup Generator

A modern web tool to quickly rebuild your Linux environment by selecting apps and generating installation scripts based on your distribution.

No more remembering packages after formatting your machine — just select, generate, and run.

---

## ✨ Features

- 🧩 Multi-distro support (APT, Pacman, DNF, Zypper, APK)
- 📦 App selection via simple UI
- ⚡ Instant install script generation
- 💾 Export / Import setup (backup your environment)
- 🔍 Organized by categories (dev, browser, communication, gaming, etc.)
- 🧠 Smart mapping between apps and package managers
- 📜 Supports Flatpak, Snap, and custom install scripts

---

## 🎯 Purpose

BootStack was built to solve a simple problem:

> “I just formatted my machine… now what did I have installed again?”

This tool makes Linux environments reproducible, shareable, and easy to restore.

---

## 🚀 How it works

1. Select your apps from the dashboard
2. Choose your Linux distribution
3. Generate your install script
4. Copy and run it in your terminal

Optional:
- Export your setup as JSON
- Import setups from other machines or users

---

## 📦 Supported Package Managers

- APT (Debian, Ubuntu, Mint, Pop!_OS)
- Pacman (Arch, Manjaro, EndeavourOS)
- DNF (Fedora, RHEL, CentOS)
- Zypper (openSUSE)
- APK (Alpine Linux)
- Flatpak (universal apps)
- Snap (Ubuntu ecosystem)
